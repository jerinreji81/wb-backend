# WorshipBase UG Backend v68.4.1

Small backend helper for WorshipBase Ultimate Guitar import.

## Endpoints

- `GET /health`
- `GET /api/ug/search?q=amazing%20grace`
- `GET /api/ug/import?url=https://tabs.ultimate-guitar.com/tab/...`

## Render settings

- Build command: `npm install`
- Start command: `npm start`

## Notes

This package fixes the v68.4 startup crash caused by an invalid regular expression.
